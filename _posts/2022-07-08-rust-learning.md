## Rust 开发环境配置
安装Rust版本管理器rustup和Rust包管理器cargo，官方安装脚本:
```shell
curl https://sh.rustup.rs -sSf | sh
```
将环境变量设置应用到当前终端:
```shell
source $HOME/.cargo/env
```
确认安装的Rust工具链:
```shell
rustc --version
```
